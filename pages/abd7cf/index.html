# 1. 下载node

# 1.1 官网下载

打开官网 Node.js官网 (opens new window) ，选择自己的版本，这里我使用的是Windows64位的，下载.msi安装包

下载其他版本

以14.x版本为例Index of /download/release/latest-v14.x/ (nodejs.org) (opens new window)

# 1.2 中文下载

下载 | Node.js 中文网 (nodejs.cn) (opens new window)

# 2. 安装配置

# 2.1 安装node

双击运行安装包

点击next下一步

勾选同意，点击Next下一步

选择安装位置，点击Next下一步

选择安装方式，这里选择Add to PATH方式安装（自动添加到系统环境），点击Next下一步

点击Next下一步

点击Install安装

等待安装完成

安装完成，点击Finish

右键“我的电脑”-选择属性-点击高级系统设置-点击环境变量-双击系统变量的PATH

可以看到，安装完成自动配置环境变量，如果这里没有显示，可自行添加即可

# 2.2 配置修改package的全局路径和npm缓存路径

打开刚刚安装Node.js目录，创建 node_global和node_cache两个文件夹

接着打开cmd命令工具，输入两句指令（根据自己实际修改路径）

npm config set prefix "D:\Develop\front\nodejs\node_global"
npm config set cache "D:\Develop\front\nodejs\node_cache"
1
2

右键“我的电脑”-选择属性-点击高级系统设置-点击环境变量-新建系统变量

接着双击用户变量的PATH

输入 D:\Develop\front\nodejs\node_global

点击确定，配置完成。

# 2.3 配置淘宝镜像

配置cnpm，配置淘宝镜像，默认路径下载比较缓慢，配置淘宝镜像可以加速下载速度，打开命令窗口执行以下命令即可，如果不想配置cnpm，可直接跳过此步骤；

npm install -g cnpm --registry=https://registry.npm.taobao.org
1

直接设置为npm设置淘宝镜像仓库即可 😄

npm config set registry https://registry.npm.taobao.org
1

# 2.4 查看npm配置信息

npm config list
1

# 2.5 查看node版本信息

npm -v
1

# 3. 配置yarn

# 3.1 配置yarn

# 1.1 安装yarn

npm install yarn -g
1

右键“我的电脑”-选择属性-点击高级系统设置-点击环境变量-双击系统变量的PATH

# 2. 卸载

npm uninstall yarn -g  // yarn卸载
1

# 3. 查看yarn版本

yarn -v
1

# 3.2 查看node配置信息

全局路径、缓存路径、以及淘宝镜像路径，查看是否安装正确

yarn config list
1

# 3.3 查询与更换源

yarn config get registry
yarn config set registry https://registry.npm.taobao.org/
