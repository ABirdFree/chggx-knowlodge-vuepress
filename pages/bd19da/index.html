    # 配置环境-Nacos

    注意

    升级到2.0及以上版本：

    Nacos启动报错：Error creating bean with name ‘user‘: Unsatisfied dependency expressed through field ‘jwt‘ (opens new window)

    # 1. 下载安装

    下载地址：Releases · alibaba/nacos (github.com) (opens new window)

    # 2. windows安装

    启动

    windows下启动nacos (opens new window)；方式二 (opens new window)

    startup.cmd -m standalone # 单机启动命令
    1

    访问：http://localhost:8848/nacos

    1. 单机配置 (opens new window)
    ./startup.sh -m standalone # 单机启动命令
nohup /opt/develop/nacos/bin/startup.sh -m standalone &
