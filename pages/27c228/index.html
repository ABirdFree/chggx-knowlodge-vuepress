# 3. 常用命令

1. docker ps # 列出所有在运行的容器信息
2.docker images # 查看镜像 
2. docker stop 容器名  #停止nexus
3. docker start 容器名 #启动nexus 启动时间大约1分钟
4. docker restart 容器名 # 重启容器mysql
5. docker stop 容器名/容器id  # 停止容器
6. docker rm 容器名/容器id # 删除容器
7. docker rmi xxxx 删除镜像

 

1. docker inspect 容器id # 查询容器信息
2. docker exct -it 容器id/容器名 bash  #(进去容器内部)
3. Exit #退出
docker version # 查看版本
systemctl start docker # 启动docker
systemctl stop docker # 关闭停止docker
systemctl daemon-reload # 重新加载并配置(修改文件后)
systemctl restart docker # 重新加载并配置(修改文件后)
