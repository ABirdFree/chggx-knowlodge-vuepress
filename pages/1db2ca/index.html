# Visual Studio 2022 社区版

# 1. 下载

# 1.1 官网：

Visual Studio 2022 IDE - 适用于软件开发人员的编程工具 (microsoft.com) (opens new window)

# 1.2 下载免费版

# 2. 安装

双击安装包 VisualStudioSetup.exe ，点击继续。

# 2.1 安装时候的配置

  • 安装位置

  • 工作负荷：选择“使用C++的桌面开发”

  • 点击”安装“按钮

# 2.2 安装时候遇到的问题

问题： 在线下载时候显示下载速度为0kb？

解决： 修改电脑的DNS服务器地址为8.8.8.8或8.8.8.4

# 2.3. 打开注意

1、 社区版登录后免费试用，可以登陆自己的微软账号免费试用。

2、登录后，选择皮肤进入。
